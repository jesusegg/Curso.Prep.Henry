Explicacion del && (and)
es una condicional logica que permite determinar el valor boolean en un codigo para que el resultado sea true con
este operador ambas sentencias debes ser true de lo contraio el valor seria false 

Explicacion del || (or)
es una condicional logica que permite determinar el valor boolean en un codigo para que el resultado sea true con
este operador alguna de las sentencias debes ser true y solo cuando ambas sean false el resultado tambien lo seria
